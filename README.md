## Built With

* [Next.Js](https://nextjs.org/)

* [Sanity](https://www.sanity.io/)


### Getting started

1. Clone the project

```sh
$ git clone https://github.com/Nababratadeb-NITA/Artism.git
```

2. Install the dependencies:

```sh
$ yarn

# or

$ npm install
```

3. Run the project

```sh
$ npm run dev

# or to build

$ npm run build
```

## Deploy to Vercel

Use the [Vercel CLI](https://vercel.com/download)

```sh
vercel
```
